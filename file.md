# Input rule

All inputs are treated as follows:

      4       4
     / \      4 2 5
     2  5  =  2 1 3
    / \       1 -1 -1
    1 3       3 -1 -1
Example №2:

     1       3
    / \  =   1 2 3
    2  3     2 -1 -1
             3 -1 -1
